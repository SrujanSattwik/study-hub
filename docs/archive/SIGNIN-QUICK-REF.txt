🚀 START BACKEND
   cd backend
   npm start
   → Server runs on http://localhost:3001

🔐 LOGIN ENDPOINT
   POST http://localhost:3001/auth/login
   Body: { "email": "user@email.com", "password": "password" }
   
   Success Response:
   {
     "success": true,
     "token": "uuid-token",
     "user": {
       "user_id": "...",
       "full_name": "User Name",
       "email": "user@email.com",
       "role": "student"
     }
   }

📝 TEST FLOW
   1. Create account via Sign Up (existing OTP flow)
   2. Go to Login page
   3. Enter email & password
   4. Click "Sign In"
   5. Should redirect to dashboard
   6. Login button → Profile badge (initials + name)
   7. Hover profile → Logout option
   8. Click Logout → back to login
